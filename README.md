# Verbal duality database

This repository contains the database used in the study *Verbal duality in comparison with verbal plurality and its place within the domain of verbal number*.

The database brings together languages for which verbal duality has been reported in the literature. 

## Files

- `metadata.csv` — metadata for the languages in the sample, including language name, ISO code, Glottocode, genus, macro-area, coordinates, the descriptive source used for annotation, and the reference in which verbal duality in the language was reported.
- `annotation.csv` — annotation of number markers and morphological operations in the languages of the sample. Each row corresponds to one marker or morphological operation.

## Annotation

The annotation includes information on:

- domain of marking
- number value
- marker or morphological operation
- marking strategy
- lexical distribution
- productivity
- obligatoriness
- functions
- targeted participant
- additional distinctions
- source page(s)
- comments

The tag `NC` means ‘not clear’. Values followed by `?` are tentative.
